过滤器与包装器

发表于 2015-07-31   |   分类于 web   |  

过滤器作用

请求过滤器

1：完成安全检查
2：建立请求审计或日志

响应过滤器

1：压缩响应流
2：追加或修改响应流

包装器

包装器包装了实际的请求或响应对象，而且把调用委托给实际的对象，还允许你对定制请求或响应做所需的额外处理。过滤器可以用于创建请求包装器或创建响应包装器。Sun创建了4个”便利”包装器类：

DD配置

如果有多个过滤器符合，则按在DD声明的顺序执行过滤链。

注意点

1：过滤器顺序的容器规则

先找到与URL模式匹配的所有过滤器。一旦将与URL匹配的所有过滤器都放在链中，容器会用同样的方法确定与DD中匹配的过滤器。
