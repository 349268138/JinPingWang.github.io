bootstrap源码之全局样式

发表于 2015-01-09   |   分类于 bootstrap   |  

介绍了bootstrap中对html,body,以设置全局样式。

源码

剖析

html元素

font-family: sans-serif：设置字体，如果子元素没有设置字体，则默认会使用该字体。
-webkit-text-size-adjust: 100%;：以chrome为内核的浏览器，显示中文时如果font-size小于12px时，就默认用12px，使用这个属性可以修改这个操作。
-ms-text-size-adjust：As of Internet Explorer for Windows Phone 8.1 Update, Internet Explorer for Windows Phone supports “-webkit-text-size-adjust” as an alias for this property.
font-size：字体设置。
-webkit-tap-highlight-color：是一个 不规范的属性（unsupported WebKit property），它没有出现在 CSS 规范草案中。当用户点击iOS的Safari浏览器中的链接或JavaScript的可点击的元素时，覆盖显示的高亮颜色。

body元素

margin：设置外边距。
font-family：设置字体。
font-size：设置字体大小。
line-height：设置行高。
color：设置字体颜色。
background-color：设置背景颜色。

每个元素通用

box-sizing：
box-sizing属性可以为三个值之一：content-box（default），border-box，padding-box。
content-box，border和padding不计算入width之内
padding-box，padding计算入width内
border-box，border和padding计算入width之内，其实就是怪异模式了。

对不同的浏览器有不同的使用方式：
-webkit-box-sizing: border-box; // for ios-safari, android
-moz-box-sizing: border-box; //for ff
box-sizing: border-box; //for other
