这个接口是继承于Collection接口，元素是有序的，可以对指定位置进行操作。
另外要注意一点的是：此接口中的add方法与Collection中的add方法不一样，如上述源代码所说。
List接口提供了两种访问元素的协议：一种是用迭代器，另一种是用get和set方法随机地访问每个元素。迭代器这一种方式适用于链表LinkedList，而get和set方式适用于ArrayList。
