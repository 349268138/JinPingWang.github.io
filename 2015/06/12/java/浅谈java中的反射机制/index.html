浅谈java中的反射机制

发表于 2015-06-12   |   分类于 java   |  

反射

定义

能够分析类能力的程序

作用

1：运行中分析类的能力
2：运行中查看对象
3：实现通用的数组操作代码
4：利用Method对象

Class类

创建

Object类的getClass()方法

Object类中有getClass()方法会返回一个Class类型实例，所以任何继承Object类的对象都可以调用这个方法获得Class类实例。

方法

newInstance()

通过调用Class对象的newInstance()方法，就能创建一个这个Class对应类型的实例，调用的是这个类型的默认构造函数，如果没有默认构造函数则抛出异常。

getFields(), getMethods(), getConstructors()

返回类以及超类提供的public域数组，public方法数组，public构造器数组。

getField(String name), getMethod(String name, Class<?>… parameterTypes), getConstructor(Class<?>… parameterTypes)

返回类以及超类中指定相关参数的域，方法，构造器。

getDeclaredFields(), getDeclaredMethods(), getDeclaredConstructors()

返回类（不包括超类）中声明的全部域，方法和构造器

getDeclaredField(), getDeclaredMethod(), getDeclaredConstructor()

返回类（不包括超类）中指定的声明的域，方法和构造器
